Interface Web

  • L'utilisateur se connecte depuis l'IHM.

  • Page d'accueil :

    • Visualisation graphique des données des capteurs
    • Selection des cultures (corrélations avec les données des capteurs + interpolation machine learning)
    • Gestion des capteurs/systèmes contrôlables (ajouts, suppressions)

    • fonctionnalités facultatives :
      Laisser l'utilisateur déclarer les seuils d'arrosage et autres systèmes,
      Autoriser la gestion des comptes utilisateurs limités (visualisation indicateurs et/ou paramétrage)

  • L'utilisateur peut modifier ses identifiants : login/mot de passe.

  • L'utilisateur se déconnecte.
